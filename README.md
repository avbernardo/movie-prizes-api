# Movie Prizes API

RESTful API to query nominated and winning movies from the Golden Raspberry Awards Worst Movie category.

## Technologies

- Node.js
- NestJS
- TypeScript
- TypeORM
- Sql.js
- Jest
- Supertest
- Csv-parser

## Prerequisites

- Node.js 18+
- pnpm

## Install dependencies

pnpm install

## Running the Project

### Development mode

pnpm start:dev

### Build

pnpm build

### Production mode

pnpm start:prod

## Running Tests

### Integration tests

pnpm test:e2e

### With coverage

pnpm test:e2e --coverage
